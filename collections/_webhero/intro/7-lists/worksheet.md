## Questions
What's the difference between `<ul>` and `<ol>`?
{% toggle %}
`<ul>` is an unordered list.<br>
`<ol>` is an ordered list.
{% endtoggle %}

What tag do you put inside `<ul>` and `<ol>`?
{% toggle %}
`<li>`, list item tag.
{% endtoggle %}
